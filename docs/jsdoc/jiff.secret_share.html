Class: secret_share

jiff.secret_share(jiff, ready, promise, value, Zp) → {secret-share}

new secret_share(jiff, ready, promise, value, Zp) → {secret-share}

Create a new share. A share is a value wrapper with a share object, it has a unique id (per computation instance), and a pointer to the instance it belongs to. A share also has methods for performing operations.
Parameters:
Name Type Description
jiff jiff-instance the jiff instance.
ready boolean whether the value of the share is ready or deferred.
promise promise a promise to the value of the share.
value number the value of the share (null if not ready).
Zp number the modulos under which this share was created.
